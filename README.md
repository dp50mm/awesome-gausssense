### [GaussSense Scratch](https://github.com/gausstoys/awesome-gausssense/tree/master/GaussSense_Scratch)
利用Scratch的擴充機制將GaussSense的Arduino mode感測值與運算值，送至Scratch，讓中小學生可以在此基礎，做出Scratch GaussSense互動作品
